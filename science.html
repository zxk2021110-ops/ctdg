疫苗与驱虫规范

科学免疫和驱虫是保护宠物生命健康的第一道防线

🐕 犬类疫苗

  • • 核心疫苗：狂犬 + 联苗
  • • 首次免疫需打满三针
  • • 此后每年补强一针

🐱 猫类疫苗

  • • 核心疫苗：狂犬 + 猫三联
  • • 首次免疫需打满三针
  • • 此后每年补强一针

🐛 驱虫指南

体内驱虫

建议每3个月一次，经常吃生肉或外出的宠物需缩短为1个月一次。

体外驱虫

无论是否经常出门，建议每月进行一次体外滴剂，预防跳蚤、蜱虫。

🚫

绝对禁止的饮食黑名单

以下食物对猫狗有致毒/致命风险，绝对不可喂食

🍫

巧克力

含有可可碱，会导致宠物心脏衰竭和神经中毒。黑巧克力危险性最高。

🍇

葡萄与葡萄干

极易引发犬类急性肾衰竭，致死率高。即使少量也可能致命。

🧅

洋葱与大葱

破坏宠物红细胞，导致严重溶血性贫血。煮熟后仍然有毒。

🍬

木糖醇

常见于无糖口香糖中，会导致宠物血糖骤降及肝损伤。

紧急提醒

如果宠物误食以上食物，请立即联系兽医或前往最近的宠物医院！时间就是生命。

日常护理指南

让毛孩子健康快乐的日常秘诀

🛁 洗澡频率

  • • 短毛犬/猫：每2-3个月一次
  • • 长毛犬/猫：每1-2个月一次
  • • 使用宠物专用沐浴露
  • • 洗后务必彻底吹干

✂️ 毛发护理

  • • 短毛宠物：每周梳理1-2次
  • • 长毛宠物：每天梳理
  • • 换毛期增加梳理频率
  • • 定期检查皮肤状况

🦷 口腔健康

  • • 每周刷牙2-3次
  • • 使用宠物专用牙膏
  • • 提供磨牙玩具
  • • 定期检查牙齿

🐾 指甲修剪

  • • 每2-3周修剪一次
  • • 注意不要剪到血线
  • • 使用专用指甲钳
  • • 修剪后给予奖励

训练技巧

科学训练，建立良好关系

🎯 基础训练原则

正向强化

用零食、抚摸、表扬奖励正确行为，而不是惩罚错误行为。

一致性

全家人使用相同的口令和规则，避免宠物混淆。

耐心重复

每天短时间训练（5-10分钟），多次重复效果更好。

循序渐进

从简单指令开始，逐步增加难度。

📝 常用指令训练

  • 坐下：手持零食在宠物鼻前向上移动，自然坐下时说"坐"并奖励
  • 握手：在坐下基础上，轻轻抬起爪子说"握手"并奖励
  • 过来：后退几步，兴奋地说"过来"，宠物跑来时给予大量奖励
  • 等待：让宠物坐下，伸出手掌说"等待"，逐渐增加等待时间

常见疾病预防

早发现、早治疗、早康复

⚠️ 需要立即就医的症状

• 持续呕吐或腹泻超过24小时
• 呼吸困难或急促
• 无法站立或行走异常
• 精神萎靡、拒绝进食
• 体温异常（正常38-39.5℃）
• 异常出血或肿块

🛡️ 预防措施

  • 定期体检：每年至少一次全面检查，老年宠物每半年一次
  • 按时疫苗：严格按照疫苗接种计划执行
  • 合理饮食：避免喂食人类食物，选择优质宠物粮
  • 适量运动：根据品种和年龄提供适当运动量
  • 环境清洁：定期清洁宠物用品和生活区域

温馨提示

发现异常请及时就医，不要自行用药。定期体检是预防疾病的最佳方式。
